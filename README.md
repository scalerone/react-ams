# react-ams
基于react做的资产管理系统，分为前后端：react目录为前端部分，server为后端部分。

### 技术栈
 - react
 - antd
 - react-router
 - react-redux
 - nodejs
 - ES6
 - express

项目借鉴和参考了其他的开源项目，感谢广大的developer
